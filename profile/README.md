# 안녕하세요, 여러분 🙌
KSDC(Korea Student Developer Club)는 국내 최고의 **학생들의 개발자 그룹**으로 커뮤니티 구축과 네트워킹, 해커톤 및 세미나 개최 등 여러 활동을 진행하고 있습니다.
초등학생, 중학생, 고등학생, 대학생과 졸업생 모두가 멤버로 참여하실 수 있습니다!
